### 📂 Мои проекты

#### 📱 SrokOk — менеджер сроков годности
*Опубликовано в RuStore | [Ссылка](https://www.rustore.ru/catalog/app/ru.srokok.app)*

Приложение для учёта продуктов в холодильнике с напоминаниями о сроках годности.

- Спроектировал архитектуру и базу данных (Room, SQLite)
- Разработал интерфейс на Jetpack Compose
- Настроил уведомления через WorkManager
- Создал виджет, голосовой ввод, подбор рецептов
- Оптимизировал производительность

#### 🎵 Playlist Maker
*[GitHub](https://github.com/AlexejMatushkin/PlaylistMaker.git)*

Приложение для создания и управления плейлистами.

- Интеграция с REST API (Retrofit, OkHttp)
- Парсинг JSON, обработка ответов
- Архитектура MVVM, Clean Architecture
